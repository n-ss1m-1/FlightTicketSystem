🎯 项目概述

基于Qt的航空订票系统是一个完整的桌面应用程序，实现了航空公司售票系统的主要功能。系统采用客户端-服务器架构，支持多用户并发操作，为旅客提供便捷的航班查询和订票服务，为管理员提供全面的系统管理功能。

主要特点

🎨 现代化界面：使用Qt框架构建美观的桌面应用程序

🔄 实时通信：基于TCP/IP协议的客户端-服务器通信

💾 数据持久化：MySQL数据库支持，确保数据安全可靠

👥 多用户支持：普通用户和管理员两级权限管理

📱 模块化设计：清晰的代码结构和功能模块划分

✨ 功能特性

👤 普通用户功能

用户管理：注册、登录、个人信息管理
航班查询：多条件灵活搜索航班信息
在线订票：选择航班、乘机人、生成订单
订单管理：支付订单、改签、退票、历史订单查询
乘机人管理：添加、删除常用乘机人信息
个人中心：修改密码、更换手机号、应用设置

🛠️ 管理员功能

用户管理：查看所有用户、添加/删除用户账户
航班管理：添加、删除、修改、查询航班信息
订单管理：查看所有订单、处理退票/改签
系统监控：实时查看在线用户、系统日志
数据管理：数据库备份与恢复

🚀 快速开始
1. 克隆仓库
git clone https://github.com/i3o8ogg/FlightTicketSystem.git
cd FlightTicketSystem

2. 安装依赖
Ubuntu/Debian
sudo apt update
sudo apt install qt6-base-dev qt6-tools-dev qt6-network-dev \
                 mysql-server mysql-client libmysqlclient-dev \
                 unixodbc unixodbc-dev

Windows
安装 Qt 6.x
安装 MySQL 8.0
安装 MySQL ODBC Driver

3. 数据库配置
# 进入SQL目录
cd sql

# 登录MySQL
mysql -u root -p

# 执行数据库脚本
source schema.sql


4. 编译项目
# 编译服务器端
cd ../FlightTicketServer
qmake FlightTicketServer.pro
make

# 编译客户端
cd ../FlightTicketClient
qmake FlightTicketClient.pro
make


5. 运行系统
# 启动服务器（先运行）
./FlightTicketServer/flight_ticket_server

# 启动客户端（可启动多个）
./FlightTicketClient/flight_ticket_client


📖 使用说明

普通用户流程

1.注册登录：首次使用需注册账户

2.查询航班：选择出发地、目的地、日期等条件

3.选择航班：从搜索结果中选择合适的航班

4.填写信息：选择乘机人或添加新乘机人

5.生成订单：确认订单信息并提交

6.支付订单：完成在线支付

7.管理订单：在个人中心查看和管理订单

管理员流程

1.管理员登录：使用管理员账户登录

2.航班管理：添加、修改、删除航班信息

3.用户管理：管理用户账户和权限

4.订单管理：处理用户订单和改签请求

5.系统监控：查看系统运行状态和日志
